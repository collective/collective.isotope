==============================================================================
collective.isotope
==============================================================================

This package provides selectable views for Collections and Folders using the
`isotope`_ javascript library.

.. _isotope: http://isotope.metafizzy.co/

.. warning:: This package is NOT READY FOR PRODUCTION IN ANY WAY!
